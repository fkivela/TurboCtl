turboctl¶

The turboctl package contains the source code for TurboCtl, and is divided into the following three subpackages:

  • telegram
    • datatypes
    • codes
    • parser
    • telegram
    • api
  • ui
    • control_interface
    • table
    • docs
    • command_line_ui
    • widgets
    • advanced_tui
    • queuefile
    • status_format
  • virtualpump
    • parameter_component
    • hardware_component
    • virtualconnection
    • virtualpump

The package also contains these auxiliary modules:

  • hacks
  • __main__

and the following variables:

turboctl.__version__ = '1.1.1.dev1'¶

The current version of the TurboCtl project.

turboctl.copyright = '2019-2024 University of Helsinki Fusor Team'¶

Automatically updating copyright information.
